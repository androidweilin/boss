项目名称：国青社区 
开发周期：5个月  
项目介绍： 国青社区是中央国家机关团工委提议，专门为国家机关青年量身打造的一个社区平台，在这里可以发布帖子，创建属于自己的专版模块，
寻找并参与自己喜欢的活动、与大家评论互动、在线报名；还可以申请在线发布活动，赢取活动奖励。 

主要技术点：  
1、 使用ViewPager+Fragment+ListView展现广场和社区中不同类型的数据列表。并为ListView添加headView，用 ViewPager实现轮播图。  
2、 对ListView列表中的数据实现下拉刷新，上拉加载更多的功能。 
3、 板块选择中采用Fragment+ListView实现不同板块类型数据的展示。 
4、 使用WebView加载服务器端html来展现具体图文混排的详情界面。 
5、 发布帖子和活动时，使用XUtils的HttpUtils实现数据提交和图片上传功能。 
6、 图片加载和显示使用二级缓存机制，避免程序OOM的出现。 
7、 使用百度地图定位用户所在位置到活动地址之间的线路规划。 

职责描述：  
1、 协助搭建系统框架，完成系统中公用模块的封装和抽取。  
2、 负责广场和社区模块中帖子和板块数据列表的展现。  
3、 广场模块中帖子的展示，分享，点赞，评论等具体功能的实现。  
4、 社区模块中板块中所有帖子的展现及该板块中新贴的发布。 
5、 发布帖子时表情的插入及图片的上传的实现。

4、 调测客户端和服务器端的接口。

Android练手项目   
项目名称：应用商店  
项目介绍： 该APP类似与豌豆荚、应用汇等应用商店，可以在该应用中下载安装不同类型的APP应用，
提供应用详细信息的介绍，并提供多平台分享功能。根据不同的专栏分类显示对应的专栏应用。 

主要技术点： 
主界面根据ViewPager + Fragment + ViewPagerIndicator来搭建界面的主体，并根据各个界面的ListView展现形式抽取顶层Activity，
顶层Adapter和顶层Holder进行列表数据的填充和显示。
界面的数据通过AsyncTask开启异步任务，发送Http请求到服务端，使用JSON进行数据交互。
主界面使用ViewPager自动播放的轮播图，左侧使用侧滑菜单显示应用的设置菜单。
应用的下载使用线程池来控制加载任务数，当同时下载的任务超过3个时，其他正在下载的应用进入等待状态。
应用的详情界面可进行多平台分享。排行界面使用自定义控件展现不同长度关键字标签，并进行跨行排列。 

职责描述：  
该应用中框架的搭建和具体逻辑的实现都有个人独立完成。 

JavaWeb项目   
项目名称：中国移动综合网络资源管理系统   
开发周期：8个月 
项目介绍：该系统是为中国移动河南省公司开发的综合资源管理系统，主要负责处理省内所有移动网络资源管理、业务开通以及流程处理。我主要参与和
负责的是业务开通系统的开发，这个系统用来处理中国移动集团客户和家庭客户的业务开通、调整、取消等流程上的业务操作，属于业务流程管理系统。 

主要技术点：  
1、 主要使用Flex框架、ActionScript脚本来实现业务操作界面，同时也使用到了JSP、JS、Jquery作为前台开发支撑；
2、 后台整合 Spring、Sturts2、Hibernate 框架进行后台业务逻辑实现；  
3、 使用 Web Service 与外系统交互，使用Hessian 与内系统接口服务进行交互； 
4、 底层数据库使用Oracle 10g；  
5、 流程引擎使用普元流程引擎，主要包括流程图的绘制，发布流程模版，与流程服务器进行交互等。 
6、 在流程生成统计报表数据的开发中还使用到了任务调度框架Quartz在指定时间生成统计数据。 

职责描述：  
1、负责前台操作界面开发及后台交互请求的逻辑控制开发  
2、负责流程中主要环节业务逻辑的代码实现  
3、负责部分流程流程图的绘制及流程模版的发布  
4、编写与流程引擎交互的代码实现  
5、参与系统版本的维护及部署新版本上线
